# SmartContract demo

## Tech stack
### Server
- Nodedemon
- Typescript
- ExpressJs

### Frontend
- React
- Typescript


## Dev
### Server
```
cd server
npm start
```

### Frontend
```
cd frontend
npm start
```

## Build
### Server
```
cd server
npm run build
```

### Frontend
```
cd frontend
npm run build
```
---------------------
For SERVER
First install:
```
npm install web3 ethereumjs-util ethereumjs-tx eth-lightwallet
```
Prepare new eth wallet in Ropsten Test Net with Metamask [https://metamask.io/]

New config.key.json file with content from samplekey.json
